# Idea: Evergreen Notes

> [!NOTE]
> **Summary Statement**: Evergreen notes are written to be permanent, atomic, and interconnected. They are continuously refined over time and act as the building blocks of your thinking.

## 💡 Context & Insights
The concept of "Evergreen Notes" (popularized by Andy Matuschak) is a modern evolution of the classic Zettelkasten card. Unlike typical folders of notes that sit in a dark corner of your hard drive, Evergreen notes are designed to be reused and updated.

Key principles:
1. **Atomic**: Each note should focus on exactly *one* idea or concept. This makes it easier to link to. If a note gets too long, split it!
2. **Concept-Oriented**: Write notes about *ideas* rather than *sources*. Instead of a massive note titled `Lessons from School`, make individual notes like `Active recall boosts long-term memory retention`.
3. **Interconnected**: A note is only useful if it connects to other notes. When you create a new idea, ask yourself: *"What does this support? What does this contradict? What does this expand?"*

By linking notes together, you build a web of ideas (a **second brain**). When it's time to write a blog post, a report, or prepare for an exam, you don't start from scratch; you just pull the pre-linked cards from your vault!

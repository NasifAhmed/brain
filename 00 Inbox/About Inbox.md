# 📥 About the Inbox

Welcome to your **Inbox**! 

This is the friction-free landing pad for your Second Brain. 

### 💡 Why do we need an Inbox?
When an idea strikes, when you find an interesting article, or when someone tells you a book recommendation, **do not waste time thinking about where it belongs.**
- Open this vault.
- Create a new note in `00 Inbox/`.
- Dump the thought/URL/text immediately.
- Get back to what you were doing.

### 🧹 The Weekly Sweep
An inbox is only useful if it gets cleared out periodically. Once a week (e.g., Sunday evening):
1. Go through each note in this `00 Inbox` folder.
2. Decide if it belongs in:
   - **`20 Brain`** (as a permanent idea, concept, or source note).
   - **`30 Projects`** (as a new project or task list).
   - **`50 Archive`** (if it was a fleeting thought that is no longer useful but you want to keep).
   - **Trash** (delete if you no longer need it).
3. Move the note to its new home and apply a template if appropriate!

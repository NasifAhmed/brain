# 🧠 Welcome to Your Second Brain

Welcome! This is a clean, minimal, and friction-free digital garden designed specifically for Obsidian. 

Many "Second Brain" setups are way too convoluted, forcing you to spend more time organizing, tagging, and filing than actually **thinking** and **learning**. 

This vault is built on a simple philosophy: **Frictionless Capture, Organic Connection.**

---

## 📂 The 6-Folder Structure
To keep your active thinking separated from static reference and clutter, the vault is divided into six numbered folders:

1. **`00 Inbox/`**: The landing pad. Whenever you are in a rush, dump ideas, web links, or raw text here. Don't worry about templates or formatting—just capture it and go.
2. **`10 Daily/`**: A chronological record of your days. Use it for journaling, quick logs, habit tracking, and logging what notes you worked on today.
3. **`20 Brain/`**: The core of your knowledge base. This is where your permanent ideas (**[[Idea - Evergreen Notes]]**) and summaries of books, articles, or courses (**[[Source - How to Take Smart Notes]]**) live.
4. **`30 Projects/`**: Goal-oriented action hubs. If a task has a clear deadline and a defined "done" state, it lives here (e.g. **[[Project - Design My Second Brain]]**).
5. **`40 Templates/`**: Blueprints for creating consistent notes with one click.
6. **`50 Archive/`**: A safe storage space. When projects end or notes become inactive, drag them here to keep your daily space clean.

---

## ⚡ The Power of Connecting (Not Filing!)
In a traditional system, you put a file in *one* folder and hope you remember where it is. In Obsidian, **folders are just containers; connections are the value.**

### 1. Linking Notes
Use double brackets **`[[Note Name]]`** to link notes together. For example:
- Linking a book summary [[Source - How to Take Smart Notes]] to an idea note [[Idea - Evergreen Notes]].
- Linking a project [[Project - Design My Second Brain]] to the templates you used.

> [!TIP]
> You can create a link to a note that doesn't even exist yet! Just type `[[New Concept]]`. The link will be colored lighter. When you're ready, click it, and Obsidian will instantly create that note for you.

### 2. The Backlinks Panel
On the right-side panel of Obsidian, click the **Backlinks** icon. When you are viewing a note, it will show you *every other note in your entire vault* that links to this one. You don't have to search—connections are shown to you automatically!

### 3. The Graph View
Press `Ctrl + G` (or `Cmd + G` on Mac) to open the **Graph View**. You will see a beautiful, interactive cosmic web of your notes. As you link more notes, you'll physically watch your knowledge map grow.

## ⚙️ Initial 3-Minute Obsidian Setup
To make the most of this vault, open it in Obsidian and configure these three built-in settings:

### 1. Set Your Templates Folder
1. Go to **Settings** (gear icon bottom left) ➔ **Templates** (under Core Plugins).
2. Set **Template folder path** to `40 Templates`.
3. Set your date and time formats if you like (e.g. `YYYY-MM-DD`).

### 2. Set Your Daily Notes Folder
1. Go to **Settings** ➔ **Daily Notes** (under Core Plugins).
2. Set **New file location** to `10 Daily`.
3. Set **Template file location** to `40 Templates/Daily Note`.

### 3. Set Default Folder for New Notes
1. Go to **Settings** ➔ **Files and links**.
2. Set **Default location for new notes** to `00 Inbox`.
*This ensures any new note you create with `Ctrl + N` automatically lands in your Inbox, ready to be processed later!*

---

## 🧹 The 5-Minute Weekly Sweep
To keep this vault clean without any stress, set aside 5 minutes once a week (e.g. Sunday evening) to:
1. Open your `00 Inbox`.
2. For each note in the inbox, either:
   - Format it using a template in `40 Templates/` and move it to `20 Brain` or `30 Projects`.
   - File it under `50 Archive` if you just wanted to keep the quick clip for record.
   - Delete it if it's no longer useful.

*That's it! No complex databases, no dozens of folders. Just a clean, beautiful web of your thoughts.*
